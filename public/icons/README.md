# IronFreight app icons

These are the **canonical app icons** for IronFreight, used everywhere the product is distributed:

- **icon-192.png** (192×192) – Primary app icon. Used for:
  - Download page (`/download`)
  - Desktop app (Electron: Windows .exe, macOS .dmg, Linux)
  - PWA / web app manifest
  - Favicon and app icons when the download page is bookmarked or shared
- **icon-512.png** (512×512) – High-resolution version. Used for:
  - PWA install, splash, and store listing assets
  - Open Graph / social and link previews
  - App store submissions (when you publish to Microsoft Store, Apple App Store, etc.)

## App store submissions

When you submit to **Microsoft Store**, **Apple App Store**, or **Google Play**:

- Use **icon-192.png** and **icon-512.png** as the basis for the required sizes.
- Each store has its own size requirements (e.g. 50×50 to 1024×1024); scale or export from these assets so the shield “I” logo stays consistent.
- Do not change the design between the download site, desktop installers, and app stores—these files are the single source of truth.
